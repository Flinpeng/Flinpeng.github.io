# Git安装步骤

# 下载Git

Git官网下载链接

# 打开下载好的Git安装程序点击Next

# 选择你的安装路径

# 安装配置

# 是否创建Win开始菜单

# 之后一路Next 点击Install 开始安装

# 安装完成之后打开Git桌面快捷方式

# Git配置

# 配置你的Git邮箱和Name

git config --global user.name "xxxxx" // 设置你的Git用户名
git config --global user.email "xxxxx@xxx.com" // 设置你的Git邮箱
// 查询验证用户名和邮箱
git config --list

# 配置你的Git SSH Key

# 先查看你的"C:\Users\用户名"下是否有 .ssh文件夹

# 如果没有则执行以下命令

ssh-keygen -t rsa -C "xxxxx@xxx.com" // 输入上面设置的Git邮箱 连续三次回车即可
