# 准备工作

简介

工作之始电脑上需要这些必不可少的东西-下面是安装步骤

# Node.js

Node.js 是运行在服务端的 JavaScript，是一个基于Chrome JavaScript 运行时建立的一个平台。简单说，Node.js是JavaScript的运行环境（是运行环境，而不是JS框架），这使得JavaScript可以成为与Java，PHP，Python等语言一样，进行后台服务端的开发

# 下载Node.js

Node.js官网下载链接

# Node.js安装步骤

# 打开下载好的安装文件双击运行之后单击Next

# 勾选“I accept the terms in the License Agreement”同意许可协议,并单击Next

# Node.js默认安装路径为"C:\ProgramFiles\nodejs"

# 你可以修改你的安装路径,并单击Next(此处默认点击Next 不做任何修改)

# 此处不做任何修默认即可

# 准备就绪点击Install 开始安装

# 配置环境变量

Node.js安装完毕,默认会自动在系统的Path环境变量中配置node.exe的目录路径
打开系统环境变量/系统变量/Path 点击编辑可以看到环境变量已经配置

# 测试Node.js是否安装成功

打开cmd(Win + R)输入node -v查看Node版本信息,出现如下信息即表示安装成功 现在的Node安装会同时安装Npm 所以可以同时查看一下Npm的安装信息,输入npm -v即可
接下来安装插件测试一下
输入npm install express -g 全局安装express(-g表示全局安装)
安装成功之后我们可以到"C:\Users\Flinp\AppData\Roaming"下查看安装的插件 注:"C:\Users\Flinp\AppData\Roaming"为默认安装路径
如果需要自定义安装路径请看下面第二种方法

# 自定义Node.js全局包安装路径

安装好Node.js之后在你的nodejs安装目录中新建
node_global 和 node_cache 文件夹 打开cmd 运行下面两行命令(请选择你的文件夹路径,此处为我个人安装路径)

 npm config set prefix "C:\Program Files\nodejs\node_global"
 npm config set cache "C:\Program Files\nodejs\node_cache"

# Npm换源

因为Npm 默认是国外的访问所以下载速度可能会有些慢
可以选择更换为国内Npm淘宝镜像

npm config set registry https://registry.npm.taobao.org
//完成后可以通过下面方式来查看是否成功
npm config get registry
